# 👋 Hi, I'm Ladislav Osvald

Print production automation — building tools that remove repetitive work from prepress and studio workflows.

**order sheet → project folders → registration marks → cutter** — if it happens every day, it deserves a script.

## ⚙️ What I work on

- **Adobe scripting** — ExtendScript (ES3) tools for Illustrator: cutter registration marks, grommet marks, and batch imposition export
- **macOS workflow automation** — AppleScript dispatched via Shortcuts.app for project setup, file management, and login routines
- **Bridging the gap** between creative applications and production requirements with lightweight, focused scripts

## 🧰 Tech stack

`ExtendScript (ES3)` · `AppleScript` · `Adobe Illustrator` · `Adobe Bridge` · `Shortcuts.app` · `Shell` · `Node.js (tests)`

## 📦 Projects

**[extendscript-automation](https://codeberg.org/Osva1d/extendscript-automation)** — Adobe Illustrator toolkit for prepress: registration-mark generator for Zünd (ECHO) and Summa (OPOS) cutters, banner grommet marks, and batch PDF relink & export into imposition templates. Localised CS/EN UI, named presets, Node-based test suite. GPL-3.0.

**[applescript-automation](https://codeberg.org/Osva1d/applescript-automation)** — Print-studio workflow scripts for macOS: project folder creation from order data, Adobe Bridge header generation, and network-aware Finder setup at login. Runs via Shortcuts.app. MIT.

Both repos ship bilingual documentation (EN/CZ).
